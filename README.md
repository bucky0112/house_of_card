# House of Card (桌遊電商)

## Demo
[https://bucky0112.github.io/house_of_card/#/](https://bucky0112.github.io/house_of_card/#/)

## 簡介
這是我使用 Vue.js 開發的前端練習作品，主要功能如下：

* 前台：
  * 畫面展示（商品分類、單一商品介紹）
  * 購物車功能
  * 下單 / 填資料驗證 / 結帳
  * 優惠券使用


* 後台：
  * 管理者登入
  * 產品管理
  * 優惠券管理
  * 訂單管理
  * 圖片管理

## 使用技術

* Vue CLI 3
* Vue Router
* Vue Components
* RESTful APIs
* RWD
* Bootstrap 4
* JavaScript (ES6)
* ESlint (Airbnb)
